How the model tracks real sites

The deployed PV model is run over the last ~90 days of real weather for each known site and compared against that site's measured output. This path never runs the weather model, so it isolates PV-model accuracy from weather-forecast error.

Lines compare predicted vs actual daily energy. Red markers flag clipped (inverter-limited) days — these are excluded from the error metrics above, since the lost headroom is informative, not model error.
